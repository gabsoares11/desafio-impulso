## Como comparar municípios?

#### Como selecionar municípios para comparação? Que características selecionar para definir municípios semelhantes?

* Nessa comparação, dois municípios poderiam ser definidos como semelhantes se a taxa de mortalidade de ambos fosse semelhante


#### Descrição de duas análises ou visualizações para comparar a situação entre municípios.

* O gráfico de linha poderia ser uma das visualizações possíveis, visto que seria possível visualizar ambos os municípios e as variáveis existentes e perceber a comparação. Uma análise interessante poderia ser a visualização da correlação entre casos confirmados e mortes em determinados municípios.
